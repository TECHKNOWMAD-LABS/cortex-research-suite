We just open-sourced Cortex Research Suite -- a Python framework for AI-powered research with 27 self-evolving skills.

Built by TECHKNOWMAD LABS — a DPIIT-registered startup based in India, member of the NVIDIA Inception Program, and Zoho for Startups partner. This is our contribution to the global open-source AI infrastructure, and we are proud it is shipping from here.

**What it does:**

Research skills that do not stay static. They compete in an evaluation arena, mutate, and improve autonomously -- an implementation of the autoresearch pattern. You start an evolution run, and the next morning your research pipeline is measurably better.

**The 5-layer architecture:**

- Foundation: 27 core research skills + deterministic evaluation lab
- Evolution: Skill Organism with arena-based competition (ARENA.md)
- Data: MindSpider connector for live data ingestion and social listening
- Intelligence: Multi-source cross-referencing with contradiction detection
- Simulation: Swarm-based scenario modeling for hypothesis testing

**Why this matters for the Indian tech ecosystem:**

India has world-class engineering talent and a growing AI research community. But much of the foundational AI tooling still originates elsewhere. We want to change that. Cortex Research Suite is MIT licensed, built in Python, and designed to be extended by the community.

**A note on data handling:**

We built this with awareness of the Digital Personal Data Protection Act, 2023. The MindSpider data connector layer includes configurable data retention policies, source attribution tracking, and consent-aware data handling patterns. Research tools that process public and semi-public data need to be built with these considerations from day one, not bolted on after the fact.

**Integration:**

Works with Claude Code natively. MCP adapters for LangChain, CrewAI, and OpenAI. No vendor lock-in. Use individual skills in your existing pipeline or run the full stack.
