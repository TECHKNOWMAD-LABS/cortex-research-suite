## Add Cortex Research Suite

**[Cortex Research Suite](https://github.com/TECHKNOWMAD-LABS/cortex-research-suite)** — MLOps-oriented research toolkit with experiment tracking, data pipelines, and evaluation harnesses.

Provides skills for end-to-end ML operations: data pipeline construction, experiment tracking with hyperparameter logging, model evaluation, metric dashboards, and dependency auditing.

- ETL pipeline builder with validation
- Experiment tracker with hyperparameter and result logging
- Real-time metric dashboards
- Toolkit scanner for dependency auditing
- CI/CD integration with GitHub Actions
