## Add Cortex Research Suite

**[Cortex Research Suite](https://github.com/TECHKNOWMAD-LABS/cortex-research-suite)** — A modular AI-powered research automation toolkit built on Claude Code.

Features 27 specialized skills for research workflows including literature review, hypothesis generation, experiment tracking, and evaluation harnesses. Integrates with the ARENA trilogy (MindSpider, BettaFish, MiroFish) for knowledge synthesis, code analysis, and simulation.

Built for researchers and developers who use Claude Code as their primary development interface.

- 27 composable research skills
- TDD-enforced development with mutation testing
- Security audit rules for LLM-specific vulnerabilities
- Jupyter notebook diffing and pipeline automation
