## Add Cortex Research Suite

**[Cortex Research Suite](https://github.com/TECHKNOWMAD-LABS/cortex-research-suite)** — A Python-based research automation toolkit with 27 modular skills.

Pure Python implementation featuring mutation testing, Jupyter notebook diffing, semgrep security rules, and statistical analysis tools. Designed for Python-first research and development workflows.

- Mutation testing integration (mutmut + manual fallback)
- Jupyter notebook cell-by-cell diff engine
- Semgrep rules for Python LLM security patterns
- Statistical analysis and visualization generation
- CLI tools with argparse, JSON/text output formats
