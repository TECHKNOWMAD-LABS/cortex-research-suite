## Add Cortex Research Suite

**[Cortex Research Suite](https://github.com/TECHKNOWMAD-LABS/cortex-research-suite)** — A multi-agent research automation framework with 26 specialized AI skills.

Supports autonomous research workflows: literature review, hypothesis generation, experiment tracking, model evaluation, and report generation. Skills compose into agent pipelines via the ARENA trilogy for knowledge acquisition, code analysis, and simulation.

- 26 domain-specific agent skills
- ARENA trilogy integration (MindSpider, BettaFish, MiroFish)
- End-to-end research automation from search to publication
- Built on Claude Code with MCP server orchestration
