## Add Cortex Research Suite

**[Cortex Research Suite](https://github.com/TECHKNOWMAD-LABS/cortex-research-suite)** — Includes a configurable LLM evaluation harness with model comparison and metric dashboards.

The eval-harness and model-compare skills provide side-by-side LLM evaluation with statistical significance testing, prompt optimization feedback loops, and real-time metric tracking.

- Configurable evaluation benchmarks
- Side-by-side model comparison with statistical tests
- Prompt optimization using evaluation feedback
- Mutation testing for test suite quality assessment
- Security audit rules for LLM-specific vulnerabilities (prompt injection, unsafe deserialization)
